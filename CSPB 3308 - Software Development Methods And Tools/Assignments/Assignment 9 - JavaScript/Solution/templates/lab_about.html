About

This is a JavaScript typing game that was created for CSPB 3308.

Changes

The changes made to this game from the original assignment are:

  • Added support for the backspace button
  • Added additional challenges to the game with corresponding difficulties
  • Added a restart button
  • Added a next challenge button
  • Added a timer mechanism for the game to display the time taken to complete a challenge
  • Reset the challenge for each new challenge so only its time taken to complete is displayed
  • Added a random challenges option after the first four challenges have been completed
  • Added style changes to the HTML document with more CSS
  • Formalized the tags in the HTML document to not be in random order
