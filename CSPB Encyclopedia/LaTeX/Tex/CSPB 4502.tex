\subsection{Course Overview}

The overview of this course can be seen below.

\course{\hypertarget{CSPB:4502}{\CSPBDM} \hspace*{1pt} - Prerequisites: \hyperlink{CSPB:2270}{CSPB 2270} - Credits: 3}{
    \vspace*{-1.5em}
    \begin{highlight}[\CSPBDM]
        \subsubsection*{Brief Description of Course Content}
    
        Introduces basic data mining concepts and techniques for discovering interesting patterns hidden in large-scale data sets, focusing on issues relating to effectiveness and efficiency. Topics covered 
        include data preprocessing, data warehouse, association, classification, clustering, and mining specific data types such as time-series, social networks, multimedia, and Web data. \vspace*{1em}
        
        \subsubsection*{Specific Outcomes of Instruction}
    
        \begin{itemize}
            \item Apply the concepts and techniques of Data mining on data sets
            \item Preprocess and clean data for use in data mining
            \item Discover interesting patterns from large amounts of data
        \end{itemize}
        
        \subsubsection*{Brief List of Topics to be Covered}
    
        \begin{itemize}
            \item Data preprocessing
            \item Data warehouse 
            \item Association
            \item Classification
            \item Clustering
            \item Mining specific data types such as time-series, social networks, multimedia, and Web data
        \end{itemize}
        
        \subsubsection*{Mathematical Concepts Used}
        
        \begin{itemize}
            \item Statistics
        \end{itemize}
    \end{highlight}
}
